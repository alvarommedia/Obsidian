**Flex** en una propiedad de [[CSS]] la cual indica como debe funcionar el contenedor con respecto a sus hijos.
Se emplea en los contenedores, cambiando la propiedad [[display]] y poniéndola en **flex**.

```css
.contenedor {
	display: flex;
}
```
